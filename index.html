The line

Five engines, equal standing.

Each one earns its place the same way: it shows its work and lets you check it. Together they are the surface a person and a model build on without anyone having to be trusted.

gather

intake and witness shipped · PyPI

The research-intake organ. It reaches information anywhere, including the hard places: gated APIs, paywalls, JavaScript-walled pages, scanned PDFs, audio, facts scattered across fragments. Every item carries a provenance receipt, and a run emits a witnessed digest the other engines consume. Fair source, zero core dependencies.

The floor

The floor

Looking verified is not the same as being verifiable.

It started as a way to catch an AI in a confident lie. The same gap turned out to be everywhere: proving who you are is not permission to act, a practice run on old data is not money earned, a certificate on the wall is not safety. Something that looks verified, mistaken for the thing it is meant to guarantee.

Every engine here runs the same move. Go back to the evidence, work the answer out again, stay honest about what you could not check, and give one of three verdicts: MATCH, DRIFT, or UNVERIFIABLE. Never the word trusted, never a guess dressed as an answer. index is live now: pip install index-graph, the whole map and certificate in one command.

Build it to be checked, or do not ship it.
