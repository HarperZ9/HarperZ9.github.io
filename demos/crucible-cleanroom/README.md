# Crucible Cleanroom Demo Public Handoff

This folder is the public-ready handoff surface for the cleanroom verdict packet demo.

## Files

- `index.html` - static artifact page for public posting.
- `crucible-cleanroom-demo.zip` - downloadable demo bundle.
- `crucible-cleanroom-demo/README.md` - packet README.
- `crucible-cleanroom-demo/bundle/spec.json` - original claim spec.
- `crucible-cleanroom-demo/bundle/run.json` - witnessed run record and integrity checks.
- `crucible-cleanroom-demo/bundle/report.md` - human-readable verdict report.
- `crucible-cleanroom-demo/bundle/review.md` - cleanroom review boundary.

## Public Claim Boundary

This demo shows one tiny verifier packet that can produce `MATCH`, `DRIFT`, and `UNVERIFIABLE` outcomes from sealed records. It does not certify code, safety, compliance, or correctness beyond the demo.
