TYPEFACE · TELOS DISPLAY

Telos Display 0.6

Generated typography, built to read.

A first-party display face derived from readable outlines, then modulated by code so the public site has a native typographic signature without sacrificing scanning, mixed-case copy, or mobile delivery.

The 0.6 "crystal cut" build adds a facet pass over the derived outlines: long curve sweeps become short straight planes and selected square terminals take a small corner bevel, so letters read like cut glass at display scale and stay smooth at reading scale.

Version 0.6 · generated TTF/WOFF2 · local webfont · no external font calls

The voice is fluid, technical, and restrained.

Telos Display is not body text. It is the large-scale signature: route marks, specimen captions, short brand words, and places where the site should feel generated by the same system that draws the field art.

Systems Models Graphics Research

Synthesis, not sampling.

The new visual direction comes from the full inspiration corpus, the inspected reference implementation, and the local engine. It is not a copied skin. The site turns those signals into first-party typography, field motion, dither, symmetry, and page rhythm.

Corpus

Dark poster fields, contour density, rug-like grids, lamp symmetry, hydra tiling, acid color bursts, ASCII texture, and high-detail dither all feed the system.

Reference code

The inspected source raises the bar for document-space canvas, DPR limits, reduced-motion fallbacks, activity-aware rendering, and live interaction pulses.

Typeface

Telos Display now starts from readable outlines. Generated modulation adds identity while Kilon and Hanken Grotesk protect long-form reading.

Engine

The shared canvas synthesizes fluid washes, metaball potential, ordered dither, contour ridges, planar tiles, symmetry marks, and ASCII fields by route.

Character set.

The current public build covers the site language: uppercase, readable lowercase forms, digits, punctuation, and symbols used across route labels, specimen marks, display moments, and generated readouts.

Rebuilt forms.

The 0.6 build keeps the Kilon-derived outlines with deterministic narrowing, slant, and contour breathing, then cuts them: outer bowls carry flat crystal facets while counters stay round. Common lookalikes separate cleanly, including a dedicated lowercase l terminal, while the face stays readable at public-site scale.

How it delivers on the site.

Hero: readable display typography first, with Telos Display reserved for route marks, specimen words, and controlled brand moments.

Pages: shared nav, local webfonts, route-seeded canvas art, and specimen surfaces that connect the visual system to real work.

Body: Hanken Grotesk stays in charge of reading. Telos Display carries identity, not paragraphs.
