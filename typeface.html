TYPEFACE · TELOS DISPLAY

Telos Display is generated by the workshop.

A soft technical display face built from code, shaped for the public site, and used as a signal layer across systems, models, graphics, research, and generated media.

The 0.4 build trades the old block recipe for curved capsules, rounded joins, native lowercase forms, and optical spacing that can sit closer to the fluid field art without damaging readability.

Version 0.4 · generated TTF/WOFF2 · local webfont · no external font calls

The voice is fluid, technical, and restrained.

Telos Display is not body text. It is the large-scale signature: route marks, specimen captions, short brand words, and places where the site should feel generated by the same system that draws the field art.

Rebuilt forms.

The 0.4 build uses rounded rectangle, capsule, and oval contours with real quadratic curves. Common lookalikes still separate cleanly, but the face now leans toward soft technical movement instead of a seven-segment stencil.

How it delivers on the site.

Hero: readable display typography first, with Telos Display reserved for route marks, specimen words, and controlled brand moments.

Pages: shared nav, local webfonts, route-seeded canvas art, and specimen surfaces that connect the visual system to real work.

Body: Hanken Grotesk stays in charge of reading. Telos Display carries identity, not paragraphs.
