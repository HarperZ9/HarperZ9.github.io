# Current Story

Seventeen images, shown in the order they were made. They tell my current story.

The public sequence follows the original numbered source files from `1000119276.png` through `1000119292.png`. No interpretive titles were added. The chronology is the narrative.

The published images are 640 x 800 WebP derivatives of the 1229 x 1536 PNG sources. `manifest.json` records both source and published SHA-256 hashes, dimensions, public paths, and accessibility descriptions. The generation method is not asserted in this record.
