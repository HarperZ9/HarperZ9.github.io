# Current Story

Seventeen images, shown in the order they were made. They tell my current story.

The sequence follows the original numbered source files from `1000119276.png` through `1000119292.png`. No interpretive titles were added. The chronology is the narrative.

The public page uses one 640 x 14,368 WebP composite with 48-pixel neutral gaps between the seventeen 640 x 800 plates. `manifest.json` records the original source hashes, dimensions, accessibility descriptions, and the composite hash. The generation method is not asserted in this record.
