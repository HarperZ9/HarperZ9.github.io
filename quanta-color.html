Color, past the hex code.

Quanta Color is a Python color-science workbench for people who need more than hex conversion — perceptual spaces, HDR tone mapping, color-appearance models, ΔE metrics, chromatic adaptation, spectral utilities, and ICC profile generation, in a GUI and a CLI.

Real color science, measured and inspectable — the same instinct as the rest of the work here, pointed at light.

quanta-color 1.0.1 · MIT · on PyPI · 15+ spaces · 12 tone operators

Hex is a storage format, not a perception.

Two colors can sit a hair apart in #rrggbb and be obviously different to the eye, or be far apart in the file and look identical. Quanta Color works in the spaces that match how seeing actually behaves — CIELAB and the Oklab family for perceptual uniformity, CIECAM-style appearance for context, JzAzBz and ICtCp for HDR — and it measures difference with the metrics color scientists actually cite, not euclidean distance in the wrong space.

The question is never “what are the bytes” — it is “what will a person see,” and that is a measurement.

Don’t take the color. Measure it.

Each of these is verbatim output from the real CLI — quanta-color 1.0.1, run live.

Exhibit I Quanta Color · one color, every space that matters
$ quanta-color info ff6030
Color: #ff6030

  sRGB:       (1.0000, 0.3765, 0.1882)
  XYZ:        (0.4596, 0.2985, 0.0614)
  Lab (D65):  (61.52, 58.31, 56.98)
  Oklab:      (0.6128, 0.8175, 0.0611)
  Oklch:      (0.6128, 0.8198, 4.3)
  Luminance:  0.2984
  Dominant wavelength: 775 nm
  Contrast vs white: 3.01:1   vs black: 6.97:1

One input, resolved into the spaces that describe what the eye does — perceptual lightness, chroma, hue, luminance, even the dominant wavelength and WCAG contrast, in one call.

Captured live from quanta-color info · sRGB → XYZ → CIELAB / Oklab / Oklch with a D65 white point. The contrast figures are the same WCAG math the accessibility work on this site runs on.

Exhibit II Quanta Color · “how different” has more than one right answer
$ quanta-color difference ff0000 00ff00 --metric all
Color 1: #ff0000      Color 2: #00ff00

  CIE76         170.1404
  CIE94          74.8681
  CIEDE2000      87.6784
  CMC(2:1)      104.2193
  HyAB          201.1010

Red versus green, scored by five different ΔE formulas at once — because “perceptual difference” depends on which model you trust, and the honest answer is to show all of them.

CIE76 / CIE94 / CIEDE2000 / CMC / HyAB · the spread between them is the point: a single distance number hides the disagreement between the models.

Exhibit III Quanta Color · harmony computed in a perceptual space
$ quanta-color harmony ff6030 --scheme triadic
Base: #ff6030   Scheme: triadic

  1. #ff6030  (1.000, 0.376, 0.188)
  2. #588a00  (0.349, 0.543, 0.000)
  3. #fe51ff  (0.999, 0.320, 1.000)

A triad rotated in a hue space that respects perception — so the three read as evenly spaced to the eye, not just evenly spaced in a number.

Also: convert between any pair of 15+ spaces, spectrum from a color temperature, icc to write a display profile, and a GUI — quanta-color with no arguments.

15+ spaces, 12 tone-mappers, the metrics by name.

Spaces: sRGB, Linear RGB, XYZ, xyY, CIELAB, LCH, Oklab, Oklch, JzAzBz, JzCzhz, ICtCp, HSV, Display P3, BT.2020, Adobe RGB, ACEScg. HDR tone mapping with twelve operators — ACES (Narkowicz + Hill), AgX (neutral / punchy / golden), Reinhard, Hable/Uncharted 2, Lottes, Uchimura/Gran Turismo, PBR Neutral (Khronos glTF), BT.2390 EETF, BT.2446 Method A, and a custom knee. Difference by CIE76, CIE94, CIEDE2000, CMC, HyAB; chromatic adaptation, color-blindness simulation, gamut mapping, color naming, 3D-LUT I/O.

Color is where this whole portfolio’s habit started: trust the measurement, not the swatch.

Shipped: quanta-color 1.0.1, MIT, on PyPI, Python 3.10+, with a test suite across spaces, adaptation, difference, gamut, harmony, naming, color-blindness, and LUT I/O. Honest edge: it is a workbench and a library, not a calibrated measurement instrument — pair it with calibrate-pro for the display side.

Install it. Then ask it what you’re really seeing.

Python 3.10 or newer.

$ pip install quanta-color
$ quanta-color info ff6030            # every space for one color
$ quanta-color convert ff6030 --to oklab
$ quanta-color difference ff0000 00ff00 --metric all
$ quanta-color                         # or launch the GUI
