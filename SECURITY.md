# Security Policy

This repository hosts the public portfolio and Project Telos product surface.
It should not contain secrets, private customer material, live credentials, or
operator-only corpora.

## Reporting

Use GitHub issues or direct contact through the profile for public-site
problems such as broken links, incorrect public claims, asset issues, or
client-side security defects. Do not include secrets or private material in a
public issue.

## Scope

In scope:

- static-site build and CI behavior
- public links, assets, and metadata
- client-side JavaScript and accessibility defects
- public Project Telos presentation accuracy

Out of scope:

- private-line repos, credentials, internal corpora, or operator-only systems
- requests for live access material
- vulnerability claims that require publishing sensitive data
